## Prerequisites
- [Visual Studio](https://visualstudio.microsoft.com)
---
## Steps
1. Download MelonLoader to your game and run it once before continuing.
1. Download the [VSIX](https://github.com/TrevTV/MelonLoader.VSWizard/releases) from the GitHub repo.
1. Close all instances of Visual Studio and run the VSIX installer (double-clicking it should open it).
1. Open Visual Studio and create a new project.
1. Search for MelonLoader and click on either Mod or Plugin.
1. Enter the project info and press Create.
1. Select the EXE of the game you are modding and press Open.
1. Wait for the project creation and it should open a Visual Studio window with a working project.
